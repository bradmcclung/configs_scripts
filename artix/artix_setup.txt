Init system:
    Start with runit (good performance and documentation/support).
    Switch to dinit if runit causes too many issues.

Profile:
    Desktop; Hyprland
        Seat access: polkit

Greeter:
    greetd + gtkgreet

Network manager:
    Start with NetworkManager using iwd as the backend.
    Alternative to NetworkManager: ConnMan.
    Alternative to iwd: wpa_supplicant.

File manager:
    Start with Thunar (lightweight and fast)
    Alternative: Dolphin

Launcher:
    Start with rofi -show run (Doesn't rely on .desktop files, no DE integration required, works in minimal setups, 
        distro-agnostic, works on X11 and Wayland, has good documentation and customization).
    Alternative: fuzzel

Browser:
    Librewolf
    Alternative: Mullvad
